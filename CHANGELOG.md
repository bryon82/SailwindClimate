# Changelog

All notable changes to this project will be documented in this file.

## [v0.2.0] - 2026-07-09

### Added
- Add blending of major climate zones at the borders.
- Add GetCurrent versions of all functions in WeatherService.

### Updated
- Internal temp calculations to use Celcius instead of Farenheit.

## [v0.1.0] - 2026-07-07

### Added
- Barometer, thermometer, and hygrometer.
- Climate zones with temperature and dew point for the different regions of the game.
- A public WeatherService class which exposes many functions for other mods to retrieve various weather values.
